# 隐私政策 (Privacy Policy)

**应用名称**：大A跳水日记 (A-Share Crash Diary)
**生效日期**：2026年3月27日

欢迎您使用“大A跳水日记”App！我们非常重视您的隐私，并承诺保护您的个人信息安全。本隐私政策旨在向您说明我们的隐私实践。

## 1. 原则：完全离线，不收集任何数据

“大A跳水日记”是一款纯粹的离线单机应用。本应用**不会**在后台收集、记录、存储或上传您的任何个人信息、设备信息、地理位置或使用习惯。

## 2. 数据存储

应用内自带的所有历史暴跌数据均为本地预置公开数据。应用不需要您注册账号，所有数据和运行日志均仅留存在您的设备本地。

## 3. 网络权限

本应用仅使用 SwiftUI 和本地计算资源，不包含任何第三方广告追踪 SDK、分析及数据上报工具。本应用完全**不需要连接互联网即可完整运行**。

## 4. 您的权利

因为我们不收集您的任何数据，所以也无需针对您的数据提供删除、导出或查询功能（因为我们根本没有您的数据）。您可以随时通过删除本应用来彻底移除所有应用本体文件。

## 5. 儿童隐私

本应用不包含任何不适宜未成年人的内容，并且我们不会有意收集或请求儿童的任何信息。

## 6. 政策更新

如果我们的隐私政策未来发生变化，我们会在 App Store 的更新日志中明确提示，并更新本页面。

## 7. 联系我们

如果您对本隐私政策或应用安全性有任何疑问或反馈，可以随时在 App Store 评价页面留言，或者通过开发者邮箱与我们取得联系。

感谢您对“大A跳水日记”的信任与支持！
